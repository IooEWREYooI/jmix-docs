= 界面中使用仪表盘

如需在界面中展示仪表盘，使用 `dashboard` UI 组件。需要在界面 XML 的根元素中声明 `dashboard` 命名空间才能在界面使用仪表盘组件。

[source, xml,indent=0]
----
include::example$/ex1/src/main/resources/dashboards/ex1/screen/main/main-screen.xml[tags=namespace]
----

下面示例展示在主界面添加一个仪表盘：

[source, xml,indent=0]
----
include::example$/ex1/src/main/resources/dashboards/ex1/screen/main/main-screen.xml[tags=namespace;layout1;dashboard;layout2]
----

`dashboard` 元素有下列属性：

* `code` - 仪表盘的唯一编码。
* `jsonPath` - 仪表盘 JSON 文件的 `classPath` 路径。
* `timerDelay` - 刷新仪表盘 UI 的时间间隔，以秒为单位。
* `assistantBeanName` - Spring bean 类的引用，用于为仪表盘 xref:dashboards:adding-business-logic.adoc[添加业务逻辑]。

NOTE: 当嵌入仪表盘时，需要指定 `code` 或 `jsonPath` 属性。当两者同时有值时，`code` 的优先级高。

[[dashboard-parameter]]
可以用 `dashboard:parameter` 元素为仪表盘传参：

[source, xml,indent=0]
----
include::example$/ex1/src/main/resources/dashboards/ex1/screen/main/main-screen-example.xml[tags=dashboard]
----

`parameter` 元素有下列必需属性：

* `name` - 参数名。
* `value` - 参数值。
* `type` - 参数类型。

此外，仪表盘参数也可以在 xref:dashboards:user-interface.adoc#dashboard-parameters[仪表盘编辑] 界面创建。
