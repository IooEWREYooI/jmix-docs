= 清空实体
:page-aliases: backoffice-ui:actions/standard-actions/entity-clear-action.adoc

`EntityClearAction` 是一个实体选择器操作，用来清空 xref:vcl/components/entity-picker.adoc[EntityPicker] 的值。如果组件展示的是一对一 xref:data-model:entities.adoc[实体] 组合，实体实例也会在 `DataContext` 提交时（界面如果是实体编辑器，则是用户点击 *OK* 时）删除。

该操作通过 `io.jmix.ui.action.entitypicker.EntityClearAction` 类实现，在 XML 中需要使用操作属性 `type="entity_clear"` 定义。可以用 `action` 元素的 XML 属性定义通用的操作参数，参阅 xref:actions/declarative-actions.adoc[声明式操作] 了解细节。
