= ValueClearAction

`ValueClearAction` – это действие поля выбора значения, предназначенное для очистки значения xref:vcl/components/value-picker.adoc[ValuePicker].

Действие реализовано классом `io.jmix.ui.action.valuepicker.ValueClearAction` и объявляется в XML с помощью атрибута `type="value_clear"`. Общие свойства действий можно конфигурировать с помощью атрибутов элемента `action`, подробнее см. раздел xref:actions/declarative-actions.adoc[].

== Использование ActionPerformedEvent

Для того чтобы произвести какие-либо проверки, или взаимодействовать с пользователем перед выполнением действия, необходимо подписаться на событие `ActionPerformedEvent` действия и в нужный момент вызвать метод `execute()`.
