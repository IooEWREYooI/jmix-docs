`Accordion - 折叠面板` 是可折叠内容的容器，支持隐藏和显示大量内容。

image::vcl/containers/accordion.gif[align="center"]

组件的 XML 名称：`accordion`.

[[basics]]
== 基本用法

如果应用程序界面的空间有限或标签页的标题太长而无法显示在 xref:vcl/containers/tab-sheet.adoc[TabSheet] 中，则可以使用 `Accordion`。

`Accordion` 组件包含 `tab` 元素，表示标签页。每个 `tab` 都是一个容器，具有类似于 xref:vcl/containers/box-layout.adoc#vbox[vbox] 的垂直组件布局。

可以在 `tab` 元素内添加任意 xref:vcl/components.adoc[组件] 或 xref:vcl/containers.adoc[布局]。下面演示为每个 tab 添加一个文本标签：
