`BoxLayout` представляет собой контейнер с последовательным размещением компонентов.

XML-имена компонентов: `hbox`,`vbox` и `flowBox`.

[[basics]]
== Основы

Существует три типа `BoxLayout`:

[[hbox]]
=== hbox

`hbox` − компоненты располагаются горизонтально:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/boxlayout/box-layout-screen.xml[tags=layout-start;hbox;layout-end]
----

image::vcl/containers/hbox.png[align="center"]

[[vbox]]
=== vbox

`vbox` − компоненты располагаются вертикально, `width=100%` по умолчанию.

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/boxlayout/box-layout-screen.xml[tags=layout-start;vbox;layout-end]
----

image::vcl/containers/vbox.png[align="center"]

[[flowbox]]
=== flowbox

`flowBox` − компоненты располагаются горизонтально с переносом строк. Если в строке недостаточно места, непомещающиеся компоненты будут отображены в следующей строке.

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/boxlayout/box-layout-screen.xml[tags=layout-start;flow-box;layout-end]
----

image::vcl/containers/flow-box.png[align="center"]

[[styling]]
== Стили

`BoxLayout` можно использовать для создания расширенных составных компоновок. Например, атрибут `stylename` со значением `card` или `well` вместе с `stylename="v-panel-caption"` для закрытого контейнера сделает компонент похожим на https://vaadin.com/docs/v8/framework/layout/layout-panel[Vaadin Panel^]:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/boxlayout/box-layout-screen.xml[tags=layout-start;vaadin;layout-end]
----

image::vcl/containers/vaadin-panel.png[align="center"]
