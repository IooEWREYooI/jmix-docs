`ScrollBoxLayout - 滚动盒子布局` 是一个支持内容滚动的容器。

image::vcl/containers/scroll-box-default.gif[align="center"]

[[attributes]]
== 属性

[[scroll-bars]]
=== scrollBars

`scrollBars` 属性可以配置滚动条。支持以下值：

* `vertical` - 启用垂直滚动，默认。
* `horizontal` - 启用水平滚动。
* `both` - 启用垂直、水平双方向滚动。
* `none` - 禁止滚动。

如果布局容器内部的组件大小超出了容器的大小，则对应的滚动条会自动出现。

[[content-size]]
=== Content Size

[[content-height]]
* `contentHeight` - 设置内容高度。

[[content-width]]
* `contentWidth` - 设置内容宽度。

[[content-max-width]]
* `contentMaxWidth` - 设置内容的最大宽度。容器内容的宽度不能超过这个值，即便界面还有空间。

[source,xml,indent=0]

+

`ScrollBox` 内容可以缩小至 `contentMinWidth` 或 `contentMinHeight` 属性设置的值。

[NOTE]
====
建议设置内容的宽高，否则，放置在 `scrollBox` 中的组件只能有固定大小或默认大小。
如果没有设置内容的宽高，不要给内部组件设置 `height="100%"` 或 `width="100%"` 属性。
====
