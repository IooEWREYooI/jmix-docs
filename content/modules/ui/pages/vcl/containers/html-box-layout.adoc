= HtmlBoxLayout

++++
<div class="backoffice-ui-live-demo-container">
    <a href="https://demo.jmix.io/sampler/#main/sample?id=htmlboxlayout-simple" class="live-demo-btn" target="_blank">ДЕМОНСТРАЦИЯ</a>
</div>
++++

`HtmlBoxLayout` представляет собой контейнер, который позволяет определить расположение компонентов в HTML-шаблоне.

XML-имя компонента: `htmlBox`.

[CAUTION]
====
//todo add link to `BrowserFrame`
Не используйте `HtmlBoxLayout` для динамического контента или если вы хотите внедрить код `JavaScript`. Для этого следует использовать `BrowserFrame`.
====

[[basics]]
== Основы

Чтобы использовать шаблоны в этом контейнере, необходимо создать xref:themes/custom_theme.adoc[пользовательскую тему]. В подкаталоге пользовательской темы нужно создать каталог `layouts`, в котором будут размещаться все HTML-шаблоны. Теперь можно создать файл шаблона:

[source,html,indent=0]
----
include::example$/ex1/src/main/themes/helium-extended/layouts/sample.html[tags=html]
----

Шаблон должен содержать элементы `<div>` с атрибутами `location`. Эти элементы будут отображать определенные в XML-дескрипторе компоненты с соответствующими идентификаторами.

В файле `helium-extended.scss` настройте стили для переменных, указанных в атрибуте location файла шаблона:

[source,css,indent=0]
----
include::example$/ex1/src/main/themes/helium-extended/helium-extended.scss[tags=start;html-box-layout;end]
----

Древо проекта выглядит так:

image::vcl/containers/html-box-layout-tree.png[align="center"]

Теперь можно использовать шаблон в XML-дескрипторе экрана:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/htmlboxlayout/html-box-layout-screen.xml[tags=html]
----

image::vcl/containers/html-box-layout.png[align="center"]

[[template-contents]]
== Элемент templateContents

Также можно использовать элемент `templateContents`, чтобы напрямую задать содержимое шаблона и создать собственную компоновку. Например:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/htmlboxlayout/html-box-layout-screen.xml[tags=template-contents]
----

[source,css,indent=0]
----
include::example$/ex1/src/main/themes/helium-extended/helium-extended.scss[tags=start;template-contents;end]
----
