`HtmlBoxLayout - HTML 盒子布局` 是一个可以在 HTML 模板中定义组件位置的容器。

组件的 XML 名称：`htmlBox`。

[CAUTION]
====
不要将 `HtmlBoxLayout` 用于动态内容或嵌入 JavaScript 代码，如需要的话，请使用 xref:vcl/components/browser-frame.adoc[BrowserFrame]。
====

[[basics]]
== 基本用法

如需在布局容器中使用模板，需要创建 xref:themes/custom_theme.adoc[自定义主题]。在自定义主题的子目录下，创建 `layouts` 文件夹，这里放置所有的 HTML 模板。例如，可以创建模板如下：

[source,html,indent=0]
----
include::example$/ex1/src/main/themes/helium-extended/layouts/sample.html[tags=html]
----

模板应包含带有 `location` 属性的 `<div>` 元素。这些元素将显示 XML 中用相同 id 定义的组件。

在 `helium-extended.scss` 文件中，为模板中 `location` 属性指定的变量设置样式：

[source,css,indent=0]
----
include::example$/ex1/src/main/themes/helium-extended/helium-extended.scss[tags=start;html-box-layout;end]
----

项目树结构如下：

image::vcl/containers/html-box-layout-tree.png[align="center"]

然后可以在界面控制器中使用模板了（`htmlBox` 的 `template` 属性指定使用的模板）：

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/htmlboxlayout/html-box-layout-screen.xml[tags=html]
----

image::vcl/containers/html-box-layout.png[align="center"]
