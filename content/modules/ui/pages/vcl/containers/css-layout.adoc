`CssLayout - CSS 布局` 是一个布局容器，可以使用 CSS 完全控制这个容器里的组件的位置和样式。

组件的 XML 名称：`cssLayout`。

[[basics]]
== 基本用法

样式可以提供很多自定义功能。例如，我们创建一个布局容器，该容器可以根据界面的宽度调整内容的位置。首先，在 xref:themes/custom_theme.adoc[自定义主题] 的 CSS 文件中定义上述行为：

[source,css,indent=0]
----
include::example$/ex1/src/main/themes/helium-extended/helium-extended.scss[tags=start;css-layout;end]
----

然后可以在组件的 `stylename` 属性中使用这些样式：

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/csslayout/css-layout-screen.xml[tags=css-layout]
----

结果是，如果界面的宽度超过 `800px`，`CssLayout` 内容如下：

image::vcl/containers/css-layout-wide.png[align="center"]

结果是，如果界面的宽度小于 `800px`，`CssLayout` 内容如下：

image::vcl/containers/css-layout-narrow.png[align="center"]

[[styling]]
== 样式

`stylename` 属性还可以使用预定义的样式值，例如：

* `v-panel-caption`
* `card`
* `well`

参阅 xref:vcl/containers/box-layout.adoc#styling[BoxLayout] 相关章节示例。

还有一个特殊值：

[[v-component-group]]
* `v-component-group` - 样式用于创建组件集合，即一行无缝连接的组件：
+
[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/csslayout/css-layout-screen.xml[tags=style]
----
+
image::vcl/containers/css-layout-style.png[align="center"]
