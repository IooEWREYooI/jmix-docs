`CssLayout` – это контейнер, предоставляющий полный контроль над размещением и стилем вложенных компонентов с помощью CSS.

XML-имя компонента: `cssLayout`.

[[basics]]
== Основы

Использование стилей предоставляет множество возможностей настройки. Например, давайте создадим контейнер, который будет менять расположение своего содержимого в зависимости от ширины экрана. Для этого задайте описанное выше поведение в CSS-файле xref:themes/custom_theme.adoc[собственной темы]:

[source,css,indent=0]
----
include::example$/ex1/src/main/themes/helium-extended/helium-extended.scss[tags=start;css-layout;end]
----

Теперь вы можете использовать эти стили в атрибутах `stylename` компонентов:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/csslayout/css-layout-screen.xml[tags=css-layout]
----

Теперь если ширина экрана превышает `800px`, содержимое `CssLayout` будет выглядеть так:

image::vcl/containers/css-layout-wide.png[align="center"]

Но если ширина экрана меньше `800px`, содержимое `CssLayout` будет выглядеть так:

image::vcl/containers/css-layout-narrow.png[align="center"]

[[styling]]
== Стили

Также вы можете использовать предопределенные значения для атрибута `stylename`, например:

* `v-panel-caption`
* `card`
* `well`

См. пример использования этих значений в xref:vcl/containers/box-layout.adoc#styling[BoxLayout].

И одно особое значение:

[[v-component-group]]
* `v-component-group` - используйте этот стиль для создания сгруппированного набора компонентов. Например, чтобы создать ряд компонентов, соединенных друг с другом без пробелов:
+
[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/csslayout/css-layout-screen.xml[tags=style]
----
+
image::vcl/containers/css-layout-style.png[align="center"]
