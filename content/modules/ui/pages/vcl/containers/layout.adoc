= Layout

`layout` – это корневой компонент для других xref:ui:vcl/components.adoc[компонентов] и xref:ui:vcl/containers.adoc[контейнеров] xref:ui:screens/descriptors.adoc[дескриптора экрана]. Подобно xref:ui:vcl/containers/box-layout.adoc#vbox[vbox], расположение его компонентов вертикальное.

== Атрибуты

[[max-height]]
=== maxHeight

Задает максимальную высоту CSS для компоновки окна. Например, `"640px"`, `"100%"`.

[[max-width]]
=== maxWidth

Задает максимальную ширину CSS для компоновки окна. Например, `"640px"`, `"100%"`.

[[min-height]]
=== minHeight

Задает минимальную высоту CSS для компоновки окна. Например, `"640px", "auto"`.

[[min-width]]
=== minWidth

Задает минимальную ширину CSS для компоновки окна. Например, `"640px"`, `"auto"`.

Например:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/layout/layout-screen1.xml[tags=layout]
----

.Полноразмерный textArea в компоновке
image::vcl/containers/layout.png[align="center"]

.Полосы прокрутки появляются, когда размер окна меньше минимальных размеров компоновки
image::vcl/containers/layout-scroll.png[align="center"]

Эти атрибуты работают и в режиме диалогового окна:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/layout/layout-screen2.xml[tags=layout]
----

.Полосы прокрутки появляются, когда размер диалогового окна меньше минимальных размеров компоновки
image::vcl/containers/layout-modal.png[align="center"]
