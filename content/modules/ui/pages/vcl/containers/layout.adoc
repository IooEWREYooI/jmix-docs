= Layout 根布局

`layout` 是 xref:ui:screens/descriptors.adoc[界面描述] 中其他 xref:ui:vcl/components.adoc[组件] 和 xref:ui:vcl/containers.adoc[容器] 的根容器。与 xref:ui:vcl/containers/box-layout.adoc#vbox[vbox] 类似，内部组件垂直排布。

== 属性

[[max-height]]
=== maxHeight

设置窗口布局的最大 CSS 高度。例如，`"640px", "auto"`。

[[max-width]]
=== maxWidth

设置窗口布局的最大 CSS 宽度。例如，`"640px", "auto"`。

[[min-height]]
=== minHeight

设置窗口布局的最小 CSS 高度。例如，`"640px", "auto"`。

[[min-width]]
=== minWidth

设置窗口布局的最小 CSS 宽度。例如，`"640px", "auto"`。

示例：

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/layout/layout-screen1.xml[tags=layout]
----

.布局中占满空间的 textArea
image::vcl/containers/layout.png[align="center"]

.当窗口尺寸小于布局的最小尺寸时展示滚动条
image::vcl/containers/layout-scroll.png[align="center"]

这些属性也支持对话框模式：

[source,xml,indent=0]
----

----

.当对话框尺寸小于布局的最小尺寸时展示滚动条
