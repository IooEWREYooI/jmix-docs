`GroupBoxLayout - 分组框布局` 是一个容器，可以将一些组件框在一个区域并设置一个整体的标题。另外，这个区域还支持折叠。

image::vcl/containers/group-box-default.png[align="center"]

组件的 XML 名称：`groupBox`。

[[basics]]
== 基本用法

示例：

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/containers/groupboxlayout/group-box-layout-screen.xml[tags=group-box-default]
----

默认情况下，`groupBox` 容器的宽度是 100%，与 `xref:vcl/containers/box-layout.adoc#vbox[vbox]` 类似。

[[attributes]]
== 属性

[[caption]]
* `caption` - 定义组标题。

[[orientation]]
* `orientation` - 定义组件放置的方向 - 水平或垂直。默认值为 `vertical（垂直）`。水平摆放如下所示：
+
image::vcl/containers/group-box-horizontal.png[align="center"]

[[collapsable]]
* `collapsable` - 如果该值设置为 `true`，可以使用左上角的按钮将组件的内容隐藏。
+
image::vcl/containers/group-box-collapsable.gif[align="center"]

[[collapsed]]
* `collapsed` - 如果设置为 `true`，初始状态下组件内容会被折叠。`collapsed` 属性在 `collapsable="true"` 有效。

[[show-as-panel]]
* `showAsPanel` - 如果设置为 `true`，该组件看起来就会像 https://vaadin.com/docs/v8/framework/layout/layout-panel[Vaadin Panel^]。默认值为 `false`。
+
image::vcl/containers/group-box-panel.png[align="center"]

[[outerMargin]]
* `outerMargin` - 设置 `groupBox` 边框的外边距。如果设置为 `true`，组件的所有边都会添加外边距。要单独设置每一边的外边距，请为 `groupBox` 的每一边设置 `true` 或 `false`：
+
