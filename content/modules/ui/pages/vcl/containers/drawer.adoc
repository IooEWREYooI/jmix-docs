`Drawer` представляет собой сворачиваемую панель, которая позволяет пользователям переключать содержимое. Обычно он используется как контейнер для компонента xref:vcl/components/side-menu.adoc[SideMenu].

image::vcl/containers/drawer.gif[align="center"]

XML-имя компонента: `drawer`.

[[basics]]
== Основы

Определение `Drawer` в дескрипторе экрана должно иметь следующую структуру: xref:vcl/containers/css-layout.adoc[CssLayout] содержит и `drawer`, и остальные части компоновки экрана (например, `workArea`), с назначенным стилем `jmix-drawer-layout`.

Сам `drawer` должен включать следующие элементы, которые представляют:

* Заголовок `Drawer` –  `CssLayout` со стилем `jmix-drawer-header`.
** Значок заголовка со стилем `app-icon`.
** Сам заголовок со стилем `app-title`.
* Содержимое `Drawer`, например, xref:vcl/components/side-menu.adoc[SideMenu], со стилем `jmix-drawer-content`.
* Нижний колонтитул `Drawer` со стилем `jmix-drawer-footer`.
** xref:vcl/components/button.adoc[Кнопка] помещается в нижний колонтитул `drawer` и используется для сворачивания и разворачивания `Drawer`.

[[toggle]]
== toggle

Метод `toggle()` разворачивает или сворачивает `drawer` в зависимости от его текущего состояния. Например, если `drawer` уже свернут, он будет развернут после вызова метода `toggle()`, и наоборот.

[[is-collapsed]]
Вы также можете проверить текущее состояние `drawer` с помощью метода `isCollapsed()`.

[[expand-on-hover]]
== expandOnHover

Атрибут `expandOnHover` задает режим, при котором элемент `drawer` разворачивается при наведении. Обратите внимание, что состояние "свернут/развернут" при этом не изменяется. Значение по умолчанию – `false`.

== Пример использования

Например:

----

----

[source,java,indent=0]
----
include::example$/ex1/src/main/java/ui/ex1/screen/containers/drawer/DrawerScreen.java[tags=drawer]
----

<1> Переключение состояния `drawer` по нажатию кнопки.
<2> Изменение значка кнопки в зависимости от состояния `drawer`.
