`MultiSelectList - 多选列表` 组件在一个垂直滚动的列表中展示预定义的选项。`MultiSelectList` 支持用户选择或反选一个或多个选项，在点击选项时需要按下 *Shift* 或 *Control* 键。

组件的 XML 名称：`multiSelectList`。

== 基本用法

满足下列情况可以使用 `MultiSelectList`：

* 用户需要从选项列表中选择一项或多项。
* 选项的数量对与使用 xref:vcl/components/check-box-group.adoc[CheckBoxGroup] 来说太多了。
* 选项数量有限，且无需输入自定义值。

image::vcl/components/multiple-select-list.png[align="center"]

如需创建连接至数据的 `MultiSelectList`，使用 xref:vcl/xml.adoc#options-container[optionsContainer] 属性。示例：

== 定义选项

可以在 XML 中用 `optionsContainer`、`optionsEnum` 属性配置，或在控制器中用 `setOptionsList()`、`setOptionsMap()` 和 `setOptionsEnum()` 方法设置选项列表。

详细信息在 `SingleSelectList` 的 xref:vcl/components/single-select-list.adoc#options[定义选项] 章节说明
