= FileUploadField
:page-aliases: backoffice-ui:vcl/components/file-upload-field.adoc

`FileUploadField - 文件上传控件` 支持上传文件，并以字节数组的形式存储于实体属性。

这个控件包含标题、已上传文件的链接和一个上传按钮。当点击上传按钮的时候，会弹出系统标准的文件选择器，用户可以在这里选择需要上传的文件。

image::vcl/components/file-upload-field.gif[align="center"]

组件的 XML 名称：`fileUpload`。

[[functionality]]
== 功能

[[attributes]]
* 属性 - `FileUploadField` 的 xref:vcl/components/file-storage-upload-field.adoc#attributes[属性] 与 `FileStorageUploadField` 组件一样，除了 `fileStorage` 和 `fileStoragePutMode`。
* 方法 - `FileUploadField` 的 xref:vcl/components/file-storage-upload-field.adoc#methods[方法] 与 `FileStorageUploadField` 组件一样，除了 `getFileId()`。
* 监听器 - `FileUploadField` 的 xref:vcl/components/file-storage-upload-field.adoc#listeners[监听器] 与 `FileStorageUploadField` 组件一样。
