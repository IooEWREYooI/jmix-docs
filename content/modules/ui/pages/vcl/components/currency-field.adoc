`CurrencyField` – это разновидность xref:vcl/components/text-field.adoc[текстового поля], предназначенная для ввода денежных единиц. `CurrencyField` используется по умолчанию при генерации экранов в Studio для атрибутов, помеченных аннотацией `@CurrencyValue`. Поле содержит ярлык с обозначением валюты и по умолчанию имеет выравнивание по правому краю.

image::vcl/components/currency-field.jpg[align="center"]

XML-имя компонента: `currencyField`.

[[basics]]
== Основы

`CurrencyField` в основном повторяет функциональность xref:vcl/components/text-field.adoc[]. Вы можете вручную указать xref:vcl/components/text-field.adoc#data-type[datatype] для поля, за исключением того, что `CurrencyField` поддерживает только следующие числовые типы данных:

* `decimal`
* `int`
* `long`
* `double`
* `float`

`CurrencyField` можно привязать к xref:data/data-containers.adoc[контейнеру данных] с помощью атрибутов  `dataContainer` и `property`:

[[attributes]]
== Атрибуты

Данный компонент имеет три специфических атрибута:

[[currency]]
* `currency` - текст, который будет отображаться в ярлыке валюты.

[[currency-label-position]]
* `currencyLabelPosition` - определяет положение ярлыка внутри текстового поля. Имеет следующие возможные значения:
** `RIGHT` - справа от поля ввода (значение по умолчанию).
** `LEFT` - слева от поля ввода.

[[show-currency-label]]
* `showCurrencyLabel` - управляет видимостью ярлыка со значком валюты.
