`CurrencyField - 货币字段` 是 xref:vcl/components/text-field.adoc[] 的子类型，专门用来输入货币值。Studio 生成界面代码时，为带有 `@CurrencyValue` 注解的属性默认使用该字段。在这个字段内部有个货币符号，数字默认是右对齐状态。

image::vcl/components/currency-field.jpg[align="center"]

组件的 XML 名称：`currencyField`。

[[basics]]
== 基本用法

基本上，`CurrencyField` 跟 xref:vcl/components/text-field.adoc[] 的功能是一样的。可以给这个字段手动设置一个 xref:vcl/components/text-field.adoc#data-type[datatype]，但是只支持下列类型：

* `decimal`
* `int`
* `long`
* `double`
* `float`

`CurrencyField` 也可以通过 `dataContainer` 和 `property` 属性绑定 xref:data/data-containers.adoc[数据容器]：

[[attributes]]
== 属性

该组件有三个特殊属性：

[[currency]]
* `currency` - 定义作为货币符号的文本。

[[currency-label-position]]
* `currencyLabelPosition` - 设置组件内货币符号的位置，有下面两个值：
** `RIGHT` - 在组件文字输入的右侧（默认值）。
** `LEFT` - 在组件文字输入的左侧。

[[show-currency-label]]
* `showCurrencyLabel` - 定义是否需要显示货币符号。
