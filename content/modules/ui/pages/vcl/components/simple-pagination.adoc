= SimplePagination

`SimplePagination` – это компонент, используемый для загрузки данных по страницам. Он предназначен для использования внутри `Table`, `DataGrid`, и других компонентов-списков.

XML-имя компонента: `simplePagination`.

== Основы

`SimplePagination` имеет простой внешний вид со счетчиком строк и кнопками навигации. В отличие от xref:vcl/components/pagination.adoc[Pagination], он не может отображать страницы с определенными номерами.

Также `SimplePagination` может иметь xref:vcl/components/pagination.adoc#items-per-page[выпадающий список] для выбора количества элементов на странице.

image::vcl/components/simple-pagination.png[align="center"]

По умолчанию Jmix Studio генерирует компонент `SimplePagination` при создании для сущности экранов браузера и master-detail.

== Привязка к данным

Чтобы создать `SimplePagination`, связанный с данными, используйте вложенный элемент xref:vcl/components/pagination.adoc#container-provider[containerProvider] или xref:vcl/components/pagination.adoc#loader-provider[loaderProvider].

[NOTE]
====
Провайдеры загрузчика и контейнера не следует использовать, когда `SimplePagination` используется в `Table`, `DataGrid` и т. д., так как он получит привязку данных из компонента-списка.
====

== Автозагрузка

`SimplePagination` показывает текущее количество элементов и знак вопроса image:vcl/components/simple-pagination-question-mark.png[align="center"]. Если нажать на него, компонент загрузит общее количество элементов.

Атрибут `autoLoad` определяет, следует ли загружать количество элементов автоматически. Если установлено значение `false`, компонент не знает общего количества элементов и показывает кнопку с *[?]*. Если `autoLoad = "true"`, компонент выполняет запрос для получения общего количества элементов и отображает его. Значение по умолчанию – `false`.
