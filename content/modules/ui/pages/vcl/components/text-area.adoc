`TextArea` представляет собой многострочное поле текстового редактора.

XML-имя компонента: `textArea`.

[[basics]]
== Основы

`TextArea` в основном повторяет функциональность компонента xref:ui:vcl/components/text-field.adoc[].

----

image::vcl/components/text-area.png[align="center"]

[[attributes]]
== Атрибуты

`TextArea` имеет следующие специфические атрибуты:

* `rows` - задает количество строк текста. Значение `height` имеет приоритет над значением `rows`.
* `wordWrap` -  установите этот атрибут в `false`, чтобы отключить перенос слов.
