`TextArea - 文本域` 控件是一个多行文本编辑器。

组件的 XML 名称：`textArea`。

[[basics]]
== 基本用法

`TextArea` 与 xref:ui:vcl/components/text-field.adoc[] 组件功能基本相同。

image::vcl/components/text-area.png[align="center"]

[[attributes]]
== 属性

`TextArea` 有下列特殊属性：

* `rows` - 设置文本的行数。`height` 值的优先级高于 `rows` 的值。
* `wordWrap` - 设置该属性为 `false` 禁用文本换行。
