`ResizableTextArea` – это многострочное текстовое поле для редактирования текста с возможностью изменять размер.

image::vcl/components/resizable-text-area.gif[align="center"]

XML-имя компонента: `resizableTextArea`.

[[basics]]
== Основы

`ResizableTextArea` в основном повторяет функциональность xref:vcl/components/text-area.adoc[TextArea] и имеет единственный специфический атрибут:

[[resizable-direction]]
`resizableDirection` - определяет, каким образом пользователь может изменять размер компонента. Имеет следующие возможные значения:

* `BOTH` - значение по умолчанию. Компонент может изменять размер в обоих направлениях.
* `VERTICAL` - компонент может изменять размер только по вертикали.
* `HORIZONTAL` - компонент может изменять размер только по горизонтали.
* `NONE` - компонент не может изменять размер.

Пример использования:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/resizabletextarea/resizable-text-area-screen.xml[tags=resizable-text-area]
----

[CAUTION]
====
Обратите внимание, что компонент перестает поддерживать изменение размера в соответствующем направлении, если размер компонента задан в процентах. Например, если определить:

* `resizableDirection="VERTICAL"` и задать `height` в процентах.
* `resizableDirection="HORIZONTAL"` и задать `width"` в процентах.
* `resizableDirection="BOTH"` и задать либо `width`, либо `height` в процентах.

Во всех случаях, описанных выше, компонент не будет поддерживать изменение размера.
====
