`ResizableTextArea - 大小可调文本域` 是一个多行文本编辑器组件，大小可以调整。

image::vcl/components/resizable-text-area.gif[align="center"]

组件的 XML 名称：`resizableTextArea`。

[[basics]]
== 基本用法

`ResizableTextArea` 基本复制了 xref:vcl/components/text-area.adoc[文本域] 组件的功能，但是有一个特殊的属性：

[[resizable-direction]]
`resizableDirection` - 定义用户能以何种方式改变该组件大小。有下列可选值：

* `BOTH` - 组件可以在两个方向调整大小。默认值。
* `VERTICAL` - 组件只能在竖直方向调整大小。
* `HORIZONTAL` - 组件只能在水平方向调整大小。
* `NONE` - 组件大小不可调整。

示例：

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/resizabletextarea/resizable-text-area-screen.xml[tags=resizable-text-area]
----

[CAUTION]
====
注意，如果组件在某个方向设置的大小是百分比，则组件在该方向大小不可调整。例如，如果定义：

* `resizableDirection="VERTICAL"` 并设置 `height` 为百分比。
* `resizableDirection="HORIZONTAL"` 并设置 `width"` 为百分比。
* `resizableDirection="BOTH"` 并设置 `width` 或 `height` 为百分比。

上面列举的情况，组件均不可调整大小。
====
