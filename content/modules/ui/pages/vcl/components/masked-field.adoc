`MaskedField - 掩码字段` 是一个 xref:vcl/components/text-field.adoc[]，支持对数据的内容做限制和格式化。

组件的 XML 名称：`maskedField`。

[[basics]]
== 基本用法

`MaskedField` 功能与 xref:vcl/components/text-field.adoc[] 一样，但是对日期、社保号码、税号、电话号码等有格式的字符串输入更加方便，示例：

[[attributes]]
== 属性

[[mask]]
=== mask

`mask` 属性为字段设置掩码，可以用下列字符：

* `#` - 数字
* `U` - 大写字母
* `L` - 小写字母
* `?` - 字母
* `A` - 字母或数字
* `*` - 任何字符
* `H` - 大写十六进制字符
* `h` - 小写十六进制字符
* `~` - 加号（`+`）或减号（`-`）字符

[[value-mode]]
=== valueMode

`valueMode` 属性定义返回值的格式（带掩码或不带掩码），该属性可以有两个值：

* `clear` - 默认值，上面示例中会返回 `9967467927`。
* `masked` - 上面示例中会返回 `996-746-79-27`.
