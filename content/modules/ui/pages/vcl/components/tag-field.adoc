`TagField - 标签输入` 支持将选中的值作为标签展示。该组件基于 xref:vcl/components/suggestion-field.adoc[SuggestionField]，且功能相，最大的区别是，`TagField` 支持多选。另外还可以用 xref:vcl/components/tag-picker.adoc[TagPicker] 处理标签。

image::vcl/components/tag-field.gif[align="center"]

组件的 XML 名称：`tagField`。

[[basics]]
== 基本用法

满足下列情况可以使用 `TagField`：

* 需要将用户选择的值作为标签展示。
* 需要选取多个值。
* 选项的数量对与使用 xref:vcl/components/check-box-group.adoc[CheckBoxGroup] 来说太多了。
* 希望在数据库完成高效搜索，而不是在 UI 层加载太多数据。

== 属性

`TagField` 具有 xref:vcl/components/suggestion-field.adoc[SuggestionField] 的所有特定属性：

* xref:vcl/components/suggestion-field.adoc#async-search-delay-ms[asyncSearchDelayMs]
* xref:vcl/components/suggestion-field.adoc#min-search-string-length[minSearchStringLength]
* xref:vcl/components/suggestion-field.adoc#popup-width[popupWidth]
* xref:vcl/components/suggestion-field.adoc#suggestions-limit[suggestionsLimit]

以及一个特殊属性：

[[clear-all-visible]]
* `clearAllVisible` - 负责展示清除按钮，用于删除之前选择的所有值。默认 `clearAllVisible = "false"`。
