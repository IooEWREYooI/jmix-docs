`Slider - 滑动输入条` 是一个竖直或水平的条。通过鼠标拖拽条上的小控件可以为其设定某个范围内的一个数值。拖拽小控件同时也会显示值的大小。

image::vcl/components/slider.gif[align="center"]

组件的 XML 名称：`slider`。

[[basics]]
== 基本用法

[[datatype]]
滑动条的默认 xref:data-model:data-types.adoc[数据类型] 是 `double`。可以用组件的 `datatype` 属性设置下列数据类型：

* `int`
* `long`
* `double`
* `decimal`

用 `dataContainer` 和 `property` 属性可以将 `Slider` 与 xref:data/data-containers.adoc[数据容器] 绑定。此时，数据类型由 `property` 指定的实体属性决定。

include::example$/ex1/src/main/resources/ui/ex1/screen/component/slider/slider-screen.xml[tags=slider]

[[attributes]]
== 属性

`Slider` 组件有下列特殊属性：

[[max]]
* `max` - 数值范围的最大值，默认 100。

[[min]]
* `min` - 数值范围的最小值，默认 0。

[[orientation]]
* `orientation` - 竖直还是水平。默认值 `horizontal（水平）`。

[[resolution]]
* `resolution` - 小数点后数字的位数。默认值 0。

[[update-value-on-click]]
* `updateValueOnClick` - 定义是否可以通过点击滑动条设置值。默认值 `false`。
