Кнопка-ссылка (`LinkButton`) − это xref:vcl/components/button.adoc[кнопка], выглядящая как гиперссылка.

XML-имя компонента: `linkButton`.

Кнопка-ссылка может содержать текст или значок (или и то и другое). На изображении ниже показаны разные виды кнопок.

image::vcl/components/link-button.png[align="center"]

[[basics]]
== Основы

Пример использования:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/linkbutton/link-button-screen.xml[tags=link-button]

Кнопка-ссылка отличается от обычной кнопки `xref:vcl/components/button.adoc[Button]` только своим внешним видом. Все свойства и поведение идентичны описанным для `xref:vcl/components/button.adoc[Button]`.

[NOTE]
====

По умолчанию заголовок кнопки-ссылки разбивается на несколько строк, если по длине он превышает значение атрибута `width`. Поэтому для отображения многострочного заголовка достаточно указать абсолютное значение атрибута `width`. Если текст заголовка слишком длинный, а значение атрибута `width` не определено, то текст будет урезан.

Чтобы отобразить заголовок кнопки-ссылки в одной строке:
