`CapsLockIndicator - 大小写锁定提示` 组件在用户使用 xref:ui:vcl/components/password-field.adoc[] 输入密码的时候提示用户是否开启了大小写锁定。

组件的 XML 名称：`capsLockIndicator`。

image::vcl/components/caps-lock-indicator.png[align="center"]

[[basics]]
== 基本用法

`CapsLockIndicator` 组件需要配合 `PasswordField` 一起使用，当 `PasswordField` 获得焦点的时候该组件处理大小写锁定状态，当 `PasswordField` 失去焦点时，大小写锁定的状态就变成 inactive 了。

[[attributes]]
== 属性

* 可以用 `capsLockOnMessage` 和 `capsLockOffMessage` 属性来定义大小写锁定开启和关闭时的提示信息。
* 可以用 `visible` 属性来动态改变 `CapsLockIndicator` 的可见性。但是如果此时界面已经打开了，控制可见性就不一定能按预想工作。

[[appearance]]
== 样式
`CapsLockIndicator` 的外观可以使用带 `$jmix-capslockindicator-* prefix` 前缀的 SCSS 变量进行自定义。
