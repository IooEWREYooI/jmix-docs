`ProgressBar - 进度条` 展示一个进程的完成情况。进度可以是确定的或不定的。用 `ProgressBar` 显示需要长时间处理的任务的进度。

image::vcl/components/progress-bar-background-task.gif[align="center"]

组件的 XML 名称：`progressBar`。

[[styling]]
== 样式

[[stylename]]
要使进度条指示器显示为在进度条上移动的点（而不是增长条），请设置 `stylename` 为 `point` 值。

image::vcl/components/progress-bar-point.gif[align="center"]

默认情况下，不定进度条显示为水平状态条。要改为显示螺旋状的进度条，请设置 `stylename` 为 `indeterminate-circle` 值。

image::vcl/components/progress-bar-circle.gif[align="center"]

[[value-change-event]]
=== ValueChangeEvent

`ValueChangeEvent<Double>` 当进度值变化时发送，该事件具有下列方法：

* getValue() - 返回表示当前状态的值，在 `0.0` 和 `1.0` 之间。`1.0` 表示已经完成，`0.0` 表示还未开始。
* getPrevValue() - 返回同样的值，只不过该值表示的前一个状态。

如需以编程的方式注册事件处理器，使用组件的 `addValueChangeListener()` 方法。
