`TimeField - 时间组件` 是用来显示和输入时间的组件。它是 xref:vcl/components/date-field.adoc[] 组件的一部分，但是也可以单独使用。

image::vcl/components/time-field.png[align="center"]

组件的 XML 名称：`timeField`。

[[basics]]
== 基本用法

如需创建关联至数据的 `TimeField`，可以用 XML 的 xref:vcl/xml.adoc#data-container[dataContainer] 和 xref:vcl/xml.adoc#property[property] 属性：

[source, java,indent=0]
----
include::example$/ex1/src/main/java/ui/ex1/entity/Order.java[tags=order-starts;time;order-ends]
----

[source, xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/timefield/time-field-screen.xml[tags=data;layout-starts;time-field;layout-ends]
----

上面的例子中，界面为具有 `deliveryTime` 属性的 `Order` 实体定义了 `orderDc` 数据容器。xref:vcl/xml.adoc#data-container[dataContainer] 属性指向数据容器。xref:vcl/xml.adoc#property[property] 属性设置要显示在 `TimeField` 中的实体属性名称。

实体属性类型应该是 `java.util.Date`、`java.time.LocalTime` 或 `java.time.OffsetTime` 类型。

[[time-format]]
== 时间格式

* 时间格式通过时间数据类型定义，并且在本地化消息包中通过 `timeFormat` 键指定。
* 时间格式也可以通过 `timeFormat` 属性指定：
+
[source, xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/timefield/time-field-screen.xml[tags=time-format]
----

[[attributes]]
== 属性

[[resolution]]
=== resolution

`resolution` 属性支持定义时间精度。有下列可选值：

* `SEC`
* `MIN` - 默认值，显示至分钟
* `HOUR`

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/timefield/time-field-screen.xml[tags=resolution]
----

[[time-mode]]
=== timeMode

`timeMode` 属性选择时间是以 12 小时还是 24 小时制显示，有下列可选值：

* `H_12`
* `H_24` - 默认值
