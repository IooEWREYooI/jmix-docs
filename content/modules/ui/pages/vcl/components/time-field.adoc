`TimeField` – это поле для отображения и ввода времени. Это часть компонента xref:vcl/components/date-field.adoc[], но его также можно использовать отдельно.

image::vcl/components/time-field.png[align="center"]

XML-имя компонента:  `timeField`.

[[basics]]
== Основы

Для создания `TimeField`, связанного с данными, используйте атрибуты `dataContainer` и `property`:

[source, java,indent=0]
----
include::example$/ex1/src/main/java/ui/ex1/entity/Order.java[tags=order-starts;time;order-ends]
----

[source, xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/timefield/time-field-screen.xml[tags=data;layout-starts;time-field;layout-ends]
----

В приведенном выше примере у экрана есть контейнер данных `orderDc` для сущности `Order`, у которой есть свойство `deliveryTime`. Ссылка на контейнер данных указывается в атрибуте xref:vcl/xml.adoc#data-container[dataContainer]. Имя атрибута сущности, значение которого должно отображаться в поле, задается в атрибуте xref:vcl/xml.adoc#property[property].

Атрибут связанной сущности должен иметь тип `java.util.Date`, `java.time.LocalTime` или `java.time.OffsetTime`.

[[attributes]]
== Атрибуты

[[resolution]]
=== resolution

Атрибут `resolution` позволяет определить точность времени. Принимает следующие значения:

* `SEC`
* `MIN` - значение по умолчанию, с точностью до минуты.
* `HOUR`

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/timefield/time-field-screen.xml[tags=resolution]
----

[[time-mode]]
=== timeMode

Атрибут `timeMode` позволяет выбрать формат отображения времени, 12-часовой или 24-часовой, с соответствующими значениями:

* `H_12`
* `H_24` - по умолчанию.
