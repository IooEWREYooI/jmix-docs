`RichTextArea - 富文本域` 支持用户输入和编辑带格式的文本。工具栏提供了基本的编辑功能。`RichTextArea` 的值是一个 HTML 字符串。

image::vcl/components/rich-text-area.png[align="center"]

组件的 XML 名称：`richTextArea`。

[[basics]]
== 基本用法

`RichTextArea` 与 xref:vcl/components/text-field.adoc[TextField] 的功能一致，除了不能设置 xref:vcl/components/text-field.adoc#data-type[datatype]。因此，`RichTextArea` 仅适用于文本和 `String` 类型的实体属性。
