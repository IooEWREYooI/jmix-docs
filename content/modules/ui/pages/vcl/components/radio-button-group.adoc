= RadioButtonGroup

Компонент `RadioButtonGroup` обеспечивает выбор единственного значения из списка опций, используя группу переключателей. Если нужно выбрать несколько значений из списка, следует использовать компонент xref:vcl/components/check-box-group.adoc[CheckBoxGroup].

XML-имя компонента: `RadioButtonGroup`.

image::vcl/components/radio-button-group.png[align="center"]

[[basics]]
== Основы

[[options-enum]]
Простейший вариант использования `RadioButtonGroup` – выбор значения перечисления (enumeration) для атрибута сущности.

----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/radiobuttongroup/radio-button-group-screen.xml[tags=enum-radio-group]

В приведенном выше примере сущность `Customer` имеет атрибут `level` типа `Level`, который является перечислением.

[[functionality]]
== Функциональность

`RadioButtonGroup` имеет ту же функциональность, что и xref:vcl/components/check-box-group.adoc[CheckBoxGroup]:

* xref:vcl/components/check-box-group.adoc#options[Опции]
* xref:vcl/components/check-box-group.adoc#basics[Привязка данных]
* xref:vcl/components/check-box-group.adoc#optional-descriptions[Дополнительные описания]
* xref:vcl/components/check-box-group.adoc#attributes[Атрибуты]

Но обратите внимание, что вы сможете выбрать только одно из предложенных значений.
