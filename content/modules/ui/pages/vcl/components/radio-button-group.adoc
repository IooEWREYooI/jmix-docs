`RadioButtonGroup - 单选组` 支持使用单选按钮从选项列表中选择单个值。如需从列表中选择多个值，请用 xref:vcl/components/check-box-group.adoc[CheckBoxGroup] 组件。

组件的 XML 名称：`RadioButtonGroup`。

image::vcl/components/radio-button-group.png[align="center"]

[[basics]]
== 基本用法

[[options-enum]]

使用 `RadioButtonGroup` 最简单的场景是为 xref:data-model:entities.adoc[实体] 属性选择 xref:data-model:enumerations.adoc[enumeration] 值。

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/radiobuttongroup/radio-button-group-screen.xml[tags=enum-radio-group]
----

`Customer` 实体具有 `Level` 类型的 `level` 属性，是一个枚举类型。

[[functionality]]
== 功能

`RadioButtonGroup` 与 xref:vcl/components/check-box-group.adoc[CheckBoxGroup] 有相同的功能：

* xref:vcl/components/check-box-group.adoc#options[选项]
* xref:vcl/components/check-box-group.adoc#basics[数据绑定]
* xref:vcl/components/check-box-group.adoc#optional-descriptions[可选描述]
* xref:vcl/components/check-box-group.adoc#attributes[属性]

区别是，只能选择一个值。
