`TreeDataGrid - 树形数据网格` 是一个混合组件，拥有 xref:vcl/components/data-grid.adoc[DataGrid] 的强大功能和灵活性以及 xref:vcl/components/tree.adoc[Tree] 的层级结构。该组件用于展示包含对自身引用的 xref:data-model:entities.adoc[实体]。例如，可以用来展示产品目录结构或者公司的组织架构图。

组件的 XML 名称：`treeDataGrid`。

== 基本用法

`TreeDataGrid` 功能基本上与 xref:vcl/components/data-grid.adoc[DataGrid] 组件一致。

典型的树形数据网格如下：

image::vcl/components/tree-data-grid.png[align="center"]

`TreeDataGrid` 组件的 xref:vcl/xml.adoc#data-container[dataContainer] 属性指向一个 xref:data/collection-container.adoc[集合数据容器]。

[[hierarchy-property]]
`hierarchyProperty` 属性定义引用自身实体属性的名称。
