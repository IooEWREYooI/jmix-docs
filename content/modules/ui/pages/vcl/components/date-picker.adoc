`DatePicker - 日期组件` 是用来显示和选择日期的组件。跟 xref:vcl/components/date-field.adoc[DateField] 里面的下拉式日期选择器是有一样的外观。

image::vcl/components/date-picker.png[align="center"]

组件的 XML 名称：`datePicker`。

[[basics]]
== 基本用法

`DatePicker` 与 xref:vcl/components/date-field.adoc#basics[DateField] 的工作机制一样，除了此组件右侧没有 xref:vcl/components/time-field.adoc[TimeField]，配置也少一些。

[[attributes]]
== 属性

[[resolution]]
=== 精度

`resolution` 属性可以定义日期精度。支持下列值：

* `DAY` - 默认值，精确到天。
* `MONTH`
* `YEAR`

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/component/datepicker/date-picker-screen.xml[tags=resolution]
----

[[range-start-and-range-end]]
=== rangeStart 和 rangeEnd

可以通过这两个属性来定义可选的日期范围。一旦日期范围设定了，其他在范围之外的日期都会变成不可选状态。XML 中用 “yyyy-MM-dd” 格式设置范围，或通过对应的 setter 在 Java 中设置。

----

----
