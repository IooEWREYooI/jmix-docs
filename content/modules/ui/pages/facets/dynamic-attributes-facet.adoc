= DynamicAttributes
:page-aliases: backoffice-ui:facets/dynamic-attributes-facet.adoc

`DynamicAttributes` facet 支持 `DataGrid`、`Table` 和 `Form` 组件展示加载至关联数据容器中实体的动态属性。如果项目安装了 xref:dyn-attr:index.adoc[动态属性] 扩展组件，则可使用该 facet。

组件的 XML 名称：`dynattr:dynamicAttributes`。

[[basics]]
== 基本用法

`DynamicAttributesFacet` 在界面 XML 描述的 `facets` 元素中定义：

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/facets/dynamicattributesfacet/dynamic-attributes-facet-screen.xml[tags=namespace1;namespace2;facet]
----

然后，在 xref:dyn-attr:dynattr-managing.adoc#visibility[Visibility（可见性）] 标签页的 *Category attribute editor（属性编辑）* 中选择此界面。

参阅 xref:dyn-attr:index.adoc[] 了解更多。
