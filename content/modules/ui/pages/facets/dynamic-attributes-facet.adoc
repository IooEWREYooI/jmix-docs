= DynamicAttributes

Фасет `DynamicAttributes` позволяет компонентам `DataGrid`, `Table` и `Form` отображать динамические атрибуты сущностей, загруженных в связанные контейнеры данных. Данный фасет становится доступен при установке в проект дополнения xref:dyn-attr:index.adoc[Dynamic Attributes].

XML-имя компонента: `dynattr:dynamicAttributes`.

[[basics]]
== Основы

`DynamicAttributesFacet` определяется в элементе `facets` XML-дескриптора экрана:

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/ui/ex1/screen/facets/dynamicattributesfacet/dynamic-attributes-facet-screen.xml[tags=namespace1;namespace2;facet]
----

После этого нужно выбрать этот экран в редакторе *Category attribute editor* на вкладке xref:dyn-attr:dynattr-managing.adoc#visibility[Visibility].

Узнать больше о динамических атрибутах можно в разделе xref:dyn-attr:index.adoc[].
