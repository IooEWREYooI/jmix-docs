= tabs

`tabs` 创建一组标签页。与 xref:vc/layouts/tabSheet.adoc[tabSheet] 不同，该组件不能包含标签内容。

* XML 元素：`tabs`

* Java 类：`Tabs`

== 基本用法

`tabs` 可以有任何数量的标签页。可以按下面的方式定义：

----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/tabs/tabs-view.xml[tags=basics]
----

image::visual-components/components/tabs-basic.png[align="center", width="210"]

[TIP]
====
如需在 Jmix Studio 中添加 `tab`，在视图描述 XML 或 *Jmix UI* 层级面板中选择组件，然后点击 *Jmix UI* 组件面板的 xref:studio:screen-designer.adoc#component-inspector-add-button[Add]*->Tab* 按钮。
====

使用 <<SelectedChangeEvent,SelectedChangeEvent>> 修改页面的内容或调用其他响应标签页切换的操作。

* `HORIZONTAL` - 标签水平方式。这是默认值。
