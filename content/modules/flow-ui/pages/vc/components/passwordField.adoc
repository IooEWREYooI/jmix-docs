= passwordField 密码输入

`passwordField` 是一个用于输入密码的组件。组件中输入的字符会默认进行掩码处理。

// TODO Add a tip saying that there is also Flow UI loginForm.

* XML 元素：`passwordField`

* Java 类：`JmixPasswordField`

== 基本用法

除了没有 `datatype` 属性之外，`passwordField` 与 `textField` 基本一样。`passwordField` 的输入类型只能是 `String`。

=== preventInvalidInput

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://vaadin.com/docs/latest/components/text-field/#preventing-entry-of-invalid-characters" class="vaadin-docs-btn" target="_blank">Vaadin</a>

设置是否防止输入非法字符，合法字符的正则表达式通过 setAllowedCharPattern()` 方法设置。设置为 `true` 开启非法字符检查并阻止输入，`false` 则忽略该限制。

[[value]]
=== value

定义 `passwordField` 组件的值。

如果该值不能解析成 `String`，会展示默认的转换失败消息。
