[[disable-on-click]]
=== disableOnClick

如果 `disableOnClick` 属性设置成 `true`，这个按钮在点击之后就自动禁用，主要用来防止（意外的）多次点击。之后可以通过调用 `setEnabled(true)` 把按钮恢复成可点击状态。
