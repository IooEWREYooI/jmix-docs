`checkBox` 是一个拥有两个状态的组件：选中或者未选中。

* XML 元素：`checkBox`

* Java 类：`JmixCheckbox`

== 基本用法

默认情况下，Jmix Studio 在创建实体详情界面时，为实体的 `Boolean` 类型的属性生成 `checkBox`。

image::visual-components/components/check-box-basics.png[align="center", width="201"]

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/checkbox/check-box-view.xml[tags=data;layout;basics]
----

在 `checkBox` 元素中，`dataContainer` 属性引用 `userDc` 数据容器，`property` 属性指定实体的 `active` 属性。

设置 `checkBox` 的辅助功能标签。

[[indeterminate]]
=== indeterminate

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://vaadin.com/docs/latest/components/checkbox/#indeterminate" class="vaadin-docs-btn" target="_blank">Vaadin</a>

设置 `checkBox` 的不确定状态。

NOTE: 根据 HTML5 标准，这个状态只是影响组件的外观，并不会影响组件的值。

[[value]]
=== value

定义 `checkBox` 的值，可以是 `true` 或 `false`。
