[[layers]]
= 图层

基本上，图层分为栅格层和矢量层。栅格层由栅格图片组成，矢量层由矢量图形组成。

在地图中添加图层的方法是在 XML 描述中的 `geoMap` 元素声明 `layers` 及其配置。下面是带有一个栅格层和一个矢量层的地图示例。

[source,xml,indent=0]
----
include::example$/ex1/src/main/resources/maps/ex1/screen/order/order-edit1.xml[tags=layers-xml]
----

`selectedLayer` 是指地图主要关注的选中图层。选中的图层会触发事件、与用户点击进行交互并支持通过 UI 交互进行修改（要求图层可编辑）。

下列参数适用各种图层：

 * `id` — 必需参数，指定图层的 ID。
 * `visible` — 图层是否可见。
 * `minZoom` — 图层的最小缩放度（包含）。
 * `maxZoom` — 图层的最大缩放度（包含）。
