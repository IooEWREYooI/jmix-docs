= 股票走势图
:figure-caption!:

使用 `StockChartGroup` 组件可以创建股票走势图。

股票走势图支持多个数据集合并且有开箱即用的数据集合选择器。数据集合可以相互比较。

.多数据集合的股票走势图
image::charts/stock-chart-with-datasets.png[align="center"]

股票走势图可以在图上或者坐标轴上展示不同类型的标记。这些标记称为股票事件。

.带股票事件的股票走势图
image::charts/stock-chart-with-stockevents.png[align="center"]

股票图能支持多个股票面板。每个股票面板可以有任意数量的图。每个股票面板是个基于 xref:charts:types/serial-chart.adoc[SerialChart] 的单独序列图。

== 事件和处理器

include::ui:page$vcl/events-hanlers-generation-tip.adoc[]

* `DataSetSelectorCompareEvent` - 当数据集选择器在比较时发送。
* `DataSetSelectorSelectEvent` - 当选择数据集选择器时发送。
* `DataSetSelectorUnCompareEvent` - 当数据集选择器取消比较时发送。
* `PeriodSelectorChangeEvent` - 当使用选择器选择展示区域时发送。
* `StockChartClickEvent` - 当点击股票走势图区域时发送。
* `StockChartRightClickEvent` - 当右键点击股票走势图区域时发送。
* `StockEventClickEvent` - 当点击股票走势图标记时发送。
* `StockEventRollOutEvent` - 当鼠标移出标记时发送。
* `StockEventRollOverEvent` - 当鼠标悬停标记时发送。
* `StockGraphClickEvent` - 当点击股票图形时发送。
* `StockGraphItemClickEvent` - 当点击股票图条目时发送。
* `StockGraphItemRightClickEvent` - 当右键点击股票图条目时发送。
* `StockGraphItemRollOutEvent` - 当鼠标移出走势图条目时发送。
* `StockGraphItemRollOverEvent` - 当鼠标悬浮走势图条目时发送。
* `StockGraphRollOutEvent` - 当鼠标移出图像时发送。
* `StockGraphRollOverEvent` - 当鼠标悬浮图像中时发送。
* xref:charts:events.adoc#zoom-event[ZoomEvent]
