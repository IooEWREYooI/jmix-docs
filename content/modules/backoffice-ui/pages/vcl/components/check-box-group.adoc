= CheckBoxGroup

++++
<div class="backoffice-ui-live-demo-container">
    <a href="https://demo.jmix.io/sampler/#main/sample?id=checkboxgroup-simple" class="live-demo-btn" target="_blank">ДЕМОНСТРАЦИЯ</a>
</div>
++++

Компонент `CheckBoxGroup` обеспечивает выбор нескольких значений из списка опций, используя группу флажков (чекбоксов). Если нужно выбрать только одно значение из списка, следует использовать компонент xref:vcl/components/radio-button-group.adoc[RadioButtonGroup].

XML-имя компонента: `checkBoxGroup`.

image::vcl/components/check-box-group-operations.png[align="center"]

[[orientation]]

Атрибут `orientation` задает расположение элементов группы. По умолчанию элементы располагаются по вертикали. Значение `horizontal` задает горизонтальное расположение.

Для `CheckBoxGroup` доступны следующие предопределенные валидаторы:

* xref:vcl/miscellaneous/validator.adoc#not-null-validator[NotNullValidator]
